# Premaswini Portfolio

This is a complete GitHub-ready Vite React portfolio project.

## Included

- `package.json`
- `vite.config.ts`
- `tsconfig.json`
- `components.json`
- `server.ts`
- `index.html`
- `src/`
- `public/`
- `src/components/portfolio/` with the expected portfolio component filenames

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Upload to GitHub

Upload the contents of this folder as the repository root. Do not upload the parent folder or only the ZIP file contents nested inside another folder.

If your repository already has older Lovable upgrade files, delete old files first or overwrite them with this package. This package includes both the clean `src/router.tsx` app and a compatibility `src/components/portfolio/` folder.
